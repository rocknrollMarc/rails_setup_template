= Overview

This template bootstraps a Ruby on Rails project beyond what you get with default project generation (i.e. rails <project name>). 
This template can be applied to a fresh or existing Ruby on Rails application. That said, it might be best if you apply this
template to a fresh setup since some original files are overwritten and/or deleted.

= Features

The following highlights the changes that will be made to your Ruby on Rails app:

* Installs the {jQuery 1.4.0}[http://jquery.com] JavaScript library.
* Installs the {jQuery UI 1.7.2}[http://jqueryui.com] JavaScript library.
* Installs the {jQuery Validation 1.5.2}[http://bassistance.de/jquery-plugins/jquery-plugin-validation] JavaScript library.
* Installs the {jQuery Table Sorter 2.0.3}[http://tablesorter.com] JavaScript library.
* Installs the {jQuery Block UI 2.26}[http://github.com/malsup/blockui] JavaScript library.
* Installs the {jQuery jGrowl 1.1.2}[http://stanlemon.net/projects/jgrowl.html] JavaScript library.
* Installs the Underscore[http://github.com/documentcloud/underscore] JavaScript library.
* Removes the default Prototype/Scriptaculous libraries.
* Installs a default application.html.erb file with a basic but structured site layout.
* Installs a default {Blueprint CSS}[http://github.com/joshuaclayton/blueprint-css] theme.
* Installs a jQueryUI theme to compliment the Blueprint CSS theme.
* Installs default application graphics.
* Adds the REST[http://github.com/aeonscope/rest] gem requirement.
* Adds the LESS[http://lesscss.org] gem requirement.
* Adds the Formtastic[http://github.com/justinfrench/formtastic] gem requirement.
* Adds the {Acts as State Machine (AASM)}[http://github.com/rubyist/aasm] gem requirement.
* Adds the {Acts as List}[http://github.com/ryanb/acts-as-list] gem requirement.
* Adds the Ancestry[http://github.com/stefankroes/ancestry] gem requirement.
* Adds the {Will Paginate}[http://github.com/mislav/will_paginate] gem requirement.
* Adds the RedCloth[http://redcloth.org] gem requirement.
* Adds the Paperclip[http://www.thoughtbot.com/projects/paperclip] gem requirement.
* Adds the {RSpec}[http://github.com/dchelimsky/rspec] gem requirement.
* Adds the {RSpec for Rails}[http://github.com/dchelimsky/rspec-rails/tree/master] gem requirement.
* Adds the Cucumber[http://github.com/aslakhellesoy/cucumber/tree/master] gem requirement.
* Adds the Webrat[http://gitrdoc.com/brynary/webrat/tree/master] gem requirement.
* Adds the {Metric Fu}[http://metric-fu.rubyforge.org] gem requirement.
* Adds the Railroad[http://railroad.rubyforge.org] gem requirement.
* Adds the Bullet[http://github.com/flyerhzm/bullet] gem requirement.
* Installs the More[http://github.com/cloudhead/more] plugin.
* Installs the {Custom Error Message}[http://github.com/gumayunov/custom-err-msg] plugin.
* Installs the {Blue Ridge}[http://github.com/relevance/blue-ridge] plugin.
* Installs the {Rails Indexes}[http://github.com/eladmeidar/rails_indexes] plugin.
* Installs default configuration initializers.
* Installs a rake script for building Railroad diagrams.
* Installs a rake script that enhances seeding/reseeding of database data.
* Installs a namespaced visitor controller for default, non-secure public viewing.
* Installs default view helpers.
* Installs OmniGraffle documentation for site layout and structure.
* Provides site navigation with dynamic controller detection and menu selection.
* Provides nested drop down menu support as inspired by {Simple jQuery Dropdowns}[http://css-tricks.com/simple-jquery-dropdowns].

= Requirements

* MacOS/UNIX (might work on Windows but haven't tested).
* Ruby Gems 1.3.x or higher.
* Applied the {Ruby Gem Setup}[http://github.com/aeonscope/ruby_gem_setup] script or have manually installed all gems listed above.

= Usage

Open a terminal window and execute one the following command lines depending on your version preference:

Current Version (stable)

  rails demo -m http://github.com/aeonscope/rails_setup_template/raw/v1.0.0/template.rb

Master Version (possibly unstable, use at your own risk)

  rails demo -m http://github.com/aeonscope/rails_setup_template/raw/master/template.rb

= Notes

It is recommended that you move the following gem requirements from the config/environment.rb to the config/environments/test.rb
file after running the template:

* rspec
* rspec-rails
* cucumber
* webrat
* metric_fu

...and move the following gem requirements from the config/environment.rb to the config/environments/development.rb:

* railroad
* bullet
